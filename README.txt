程序说明：
	编译环境：python  2.7
	操作系统：windows
	依赖包：httplib2，Django
		请将install.bat文件移动到Python27的安装主目录，然后双击运行
目录结构
|─server.bat启动服务器，默认端口8000
|─sqlite3.db数据库文件
├─install.bat安装依赖插件
├─autotest 主目录
├─common_static 主目录
├─config 配置文件目录
├─interapp 主目录
├─templates 主目录
└─utils 主目录


启动服务器后，访问 http://localhost:8000/static/index.html 开启测试之旅!
